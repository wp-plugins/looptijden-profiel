Met de Looptijden.nl plugin is het mogelijk om looptijden.nl informatie te tonen zoals (profiel-, wedstrijd-, pr informatie) op je eigen website

== Description ==

Nu kan iedereen met een wordpress website eenvoudig zijn profiel van looptijden.nl presenteren op zijn website. Je kan zelf bepalen wat er wel en niet getoond dient te worden. De keuze bestaat uit "Algemeen profiel", "Persoonlijke records", "Laatste gelopen wedstrijden" of "Geplande wedstrijden". Uiteraard kan je zelf het design aanpassen door je eigen stylesheets te gebruiken.

We hopen met deze looptijden plugin een goede stap te zetten voor mensen om hun looptijden.nl profiel te kunnen promoten aan vrienden en kennissen. Mochten er vragen zijn kijk dan eerst even onder de "Hulp" sectie of je vraag er niet al bij staat. Indien dat niet het geval is kan je contact opnemen met bjorn@looptijden.nl. Ook kan je hier terecht voor verzoekjes om de Looptijden Profiel plugin verder te verbeteren en of uit te breiden met nieuwe opties.

== Installation ==

1. Download de zipfile
1. Pak deze uit in de `/wp-content/plugins/` directorie
1. Activeer de plugin in het 'Plugins' menu in WordPress
1. Plaats de looptijden profiel widget waar je deze hebben wilt
1. Voer de looptijden profiel code in
1. Of plaats op een pagina de volgende shortcut [looptijden code="xxxxxxxxx"]

== Frequently Asked Questions ==

= Hoe toon ik de Looptijden.nl plugin op mijn website? =

Je kan de plugin als widget gebruiken. Ga daarvoor naar het widget onderdeel van Wordpress. Hier kan je de Looptijden.nl widget slepen naar een lokatie waar je de widget wilt tonen. Je kan vervolgens per widget opgeven welk profiel deze moet tonen doormiddel van het invoeren van de juiste profielcode.
Of je kan in je bericht een zogeheten shortcut gebruiken die er zo uit ziet [looptijden code=xxxxxxxx]

= Waar kan ik mijn profielcode vinden? =

Login op Looptijden.nl en onder "wijzig je profiel" staat aan de onderkant je "unieke profielcode" vermeld. Kopieer deze in de Widget en je bent klaar voor gebruik.

= Waar kan in mijn instellingen aanpassen? =

Als je gaat naar de Looptijden instellingen zie je een overzicht van alle instelbare mogelijkheden van de Looptijden.nl plugin. Zoals het aan en uitzetten van onderdelen of het wel of niet tonen van gehele categorieën.

= Hoeveel profielen/widgets kan ik tegelijk tonen? = 

Je kan een onbeperkt aantal widgets tegelijk tonen hier zit geen maximum aanvast.
